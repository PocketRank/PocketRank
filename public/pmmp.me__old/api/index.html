getServer

GET /api/getServer/address[:port]
getServer 는 주소와 포트로 서버 인스턴스를 불러옵니다.
포트가 19132인 경우에는
/api/getServer/pmmp.me
와 같이 요청할 수 있습니다.
포트가 19132가 아닐 경우에는
/api/getServer/pmmp.me:포트
와 같이 요청해야 합니다.
인자 값 필수 여부
address 서버 도메인 (아이피) 필수
port 서버 포트 필수 X

getOnlineServers

GET /api/getOnlineServers/
getOnlineServers 는 현재 온라인인 모든 서버를 반환합니다.
인자 값 필수 여부
인자가 필요하지 않습니다.

getOfflineServers

GET /api/getOfflineServers/
getOfflineServers 는 현재 오프라인인 모든 서버를 반환합니다.
인자 값 필수 여부
인자가 필요하지 않습니다.

getPlayerCounts

GET /api/getPlayerCounts/address[:port]
getPlayerCounts 는 등록된 서버를 매분마다 기록했던 동접을 CSV 형태로 반환합니다.
D3 라이브러리를 사용하는 경우에
d3.csvParse(response['result']) 와 같은 형태로 사용할 수 있습니다.
인자 값 필수 여부
address 서버 도메인 (아이피) 필수
port 서버 포트 필수 X

Server Instance

MCBE RANK 는 아래와 같은 서버 데이터를 저장하고 있습니다.
이용자분들은 API를 통해 서버 데이터를 불러올 수 있습니다.
인자 값
lastOnline 마지막으로 온라인이였던 시간을 반환합니다.
player_count 현재 플레이어 접속 수 입니다.
서버가 오프라인인 경우 -1 이 됩니다.
daily 해당 서버에 대한 일간 기록입니다. 매일 자정에 초기화됩니다.
daily.high 일간 기록: 최고 동접
daily.start 일간 기록: 초기화된 시간 (매 자정)
daily.totalPing 일간 기록: 서버에 데이터 요청한 전체 횟수
daily.successPing 일간 기록: 서버에 데이터 요청하여 성공한 횟수
daily.successPercentage 일간 기록: 서버 업타임 (계산식: daily.successPing / daily.totalPing)
weekly 해당 서버에 대한 주간 기록입니다. 월요일-일요일을 한주로 취급합니다.
weekly.high 주간 기록: 최고 동접
weekly.start 주간 기록: 초기화된 시간 (월요일 0시)
weekly.totalPing 주간 기록: 서버에 데이터 요청한 전체 횟수
weekly.successPing 주간 기록: 서버에 데이터 요청하여 성공한 횟수
weekly.successPercentage 주간 기록: 서버 업타임 (계산식: daily.successPing / daily.totalPing)
total 해당 서버에 대한 전체 기록입니다.
total.high 전체 기록: 최고 동접
total.totalPing 전체 기록: 서버에 데이터 요청한 전체 횟수
total.successPing 전체 기록: 서버에 데이터 요청하여 성공한 횟수
total.successPercentage 전체 기록: 서버 업타임 (계산식: daily.successPing / daily.totalPing)
online 해당 서버의 온라인/오프라인 여부입니다.
rank 해당 서버의 순위입니다. (서버가 오프라인인 경우 해당 값은 존재하지 않습니다.)
players 접속 중인 플레이어의 목록입니다. 쿼리가 켜진 서버에서만 볼 수 있습니다.
plugins 서버에서 사용하는 플러그인의 목록입니다. 쿼리가 켜진 서버에서 플러그인 목록 보기를 활성화 했을 때만 볼 수 있습니다.
